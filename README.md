# Interactive Minimalist Site

Минималистичный интерактивный веб-сайт с множеством полезных функций.

## 🚀 Возможности

- **Todo List** - Управление задачами с сохранением в localStorage
- **Калькулятор** - Полнофункциональный калькулятор
- **Таймер** - Обратный отсчет времени с уведомлениями
- **Заметки** - Создание и сохранение заметок
- **Рисование** - Canvas для рисования с выбором цвета и размера кисти
- **Погода** - Получение данных о погоде для любого города

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-username/interactive-site.git
```

2. Откройте `index.html` в браузере

## 🎨 Особенности

- Минималистичный дизайн
- Адаптивная верстка для мобильных устройств
- Сохранение данных в localStorage
- Плавные анимации и переходы
- Интуитивный интерфейс

## 🛠️ Технологии

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Canvas API
- LocalStorage API
- Fetch API

## 📱 Использование

Просто откройте `index.html` в любом современном браузере. Все функции работают без необходимости установки дополнительных зависимостей.
